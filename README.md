# 多因子打板策略 (nice_try.py)

## 📋 策略概述

本策略是一个基于聚宽平台的多因子打板策略，融合了涨停板策略与多因子评分系统。策略通过五大交易模式捕捉不同市场环境下的投资机会，使用六维度评分系统进行股票筛选，并配备多层风险控制机制。

**策略来源**：
- 原始策略：[聚宽多因子策略入门](https://www.joinquant.com/post/1399)
- 优化版本：[五合一打板策略](https://www.joinquant.com/post/67584)
- 作者：JoinQuant量化课堂、好玩的豆豆

## 🎯 核心特性

### 五大交易模式

| 模式 | 优先级 | 策略逻辑 | 适用场景 |
|------|--------|----------|----------|
| **连板龙头** | 1 | 捕捉市场龙头股，连续涨停板 | 强势上涨市场 |
| **一进二** | 2 | 首板后次日高开突破 | 平稳/上涨市场 |
| **弱转强** | 3 | 昨日曾涨停但未封板，今日转强 | 震荡市场 |
| **首板低开** | 4 | 昨日涨停今日低开回踩 | 下跌市场 |
| **反向首板低开** | 5 | 昨日跌停今日高开反转 | 极端行情 |

### 六维度评分系统

策略使用六大因子对候选股票进行综合评分（总分0-30+分）：

| 因子 | 分值范围 | 评分依据 | 权重 |
|------|----------|----------|------|
| **涨停分** | 0-5分 | 连板数、一字板数、涨停时间 | 高 |
| **技术分** | 0-5分 | MA5/MA10/MA20位置、RSI、MACD | 中 |
| **放量MA分** | 0-5分 | 量比、成交量均线突破 | 中 |
| **主线分** | 0-5分 | 热点概念匹配度（本地计算） | 高 |
| **情绪分** | 0-5分 | 市场趋势、成交量、相对强度 | 低 |
| **主力资金分** | 0-5分 | 主力净流入、净占比 | 高 |

### 动态优先级调整

策略根据市场趋势自动调整模式优先级：

```python
# 强势上涨市场
优先级：连板龙头 > 弱转强 > 一进二 > 首板低开 > 反向首板低开

# 平稳市场
优先级：连板龙头 > 一进二 > 弱转强 > 首板低开 > 反向首板低开

# 下跌市场
优先级：反向首板低开 > 一进二 > 弱转强 > 连板龙头 > 首板低开
```

## 🔧 策略参数配置

### 核心参数

```python
g.position_limit = 2          # 最大持仓数量
g.min_score = 14              # 最低评分要求
g.concept_num = 8             # 热点概念最大个数
g.jqfactor = 'VOL5'           # 聚宽因子（5日平均换手率）
```

### 量比过滤范围

不同模式有不同的量比要求：

| 模式 | 量比范围 | 说明 |
|------|----------|------|
| 连板龙头 | 1.184 ~ 10.5 | 放量突破 |
| 一进二 | 1.15 ~ 6.58 | 适度放量 |
| 弱转强/首板低开 | 1.072 ~ 3.68 | 温和放量 |

### 评分阈值

```python
# 连板龙头过滤
- 主线分为0 → 过滤
- 主力资金为0且涨停分<5 → 过滤

# 一进二过滤
- 主线分为0 → 过滤
- 总分不在20-37范围 → 过滤

# 弱转强过滤
- 总分<20 → 过滤
```

## ⏰ 交易时间安排

### 定时任务调度

| 时间 | 任务 | 说明 |
|------|------|------|
| 09:25 | 盘前数据统计 | 记录市场环境、更新策略优先级 |
| 09:28 | 股票筛选 | 执行选股逻辑 |
| 09:28:10 | 早盘买入 | 执行买入操作 |
| 09:28 | 涨停低开止损 | 集合竞价止损 |
| 09:30-10:30 | 每5分钟检测 | 同花顺指标+放量大跌止损 |
| 10:31, 11:01 | 卖出检测 | 波段卖出信号检测 |
| 13:05-14:45 | 每5分钟检测 | 同花顺指标+放量大跌止损 |
| 13:31, 14:01, 14:31 | 卖出检测 | 波段卖出信号检测 |
| 14:50 | 周五建仓 | 周五尾盘建仓 |
| 15:00 | 盘后统计 | 记录收盘数据 |
| 15:05 | 交易日志 | 记录当日交易明细 |

### 周五特殊处理

- **早盘09:28:10**：仅筛选股票，不执行买入
- **尾盘14:50**：使用早盘筛选结果建仓
- **优化条件**：开盘价较现价高2%以上

## 🛡️ 风险控制机制

### 三层风控体系

#### 1️⃣ 买入前风控

- ✅ 空仓条件判断（特定日期、量能异常）
- ✅ 高风险股票排除（连续一字板≥2天）
- ✅ 量比范围限制（不同模式不同范围）
- ✅ 主线分/主力资金分过滤
- ✅ 最大持仓限制（默认2只）

#### 2️⃣ 持仓中风控

**每5分钟检测**：
- 同花顺指标波段卖信号
- 放量大跌信号（跌幅≥6%且量比≥1.5）

**紧急止损**：
- 波段卖 + 放量大跌 → 立即卖出
- 波段卖 + 亏损≥3% → 卖出
- 放量大跌 + 跌幅≥8% → 卖出

#### 3️⃣ 卖出逻辑优化

**涨停低开止损**（集合竞价）：
- 低开-3%~-1% + 市值50-100亿 → 卖出
- 低市盈率(<20) + 低开-5%~-1% → 卖出

**放量长上影卖出**：
- 上影线 > 下影线 × 3
- 开盘涨幅 < 2%
- 动态竞价量判断

- 持仓明细、交易记录

## ⚠️ 风险提示

### 策略风险

1. **涨停板策略固有风险**：
   - 追高风险大，需要严格止损
   - 流动性风险，可能无法卖出
   - 监管风险，可能被特停

2. **市场环境风险**：
   - 极端行情下策略可能失效
   - 市场风格切换时表现不佳
   - 黑天鹅事件影响

3. **技术风险**：
   - 数据延迟或错误
   - 网络故障导致无法交易
   - 策略代码bug

### 使用建议

- ✅ 先在模拟账户测试至少3个月
- ✅ 初始资金不超过总资产的10%
- ✅ 设置严格的止损线
- ✅ 定期检查策略表现
- ✅ 关注市场环境变化
